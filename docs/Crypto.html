    Crypto : Object

    Crypto provides cryptography functions. The majority of the functions are provided by crypto.subtle.
    Spec

    Instance Properties

    subtle : SubtleCrypto

    subtle provides methods to perform cryptographic operations such as encrypting, decrypting and hashing.

    Instance Methods

    getRandomValues(values : ArrayBufferView) : ArrayBufferView

    Fills values with randomly generated integers. values must be one of the integer based ArrayBufferView derivations: Int8Array, Int16Array, Int32Array, Uint8Array, Uint8ClampedArray, Uint16Array, Uint32Array. Returns values. See also Math.random().
