    new DOMMatrix(values : Iterator<Number>) : DOMMatrix

    values must have either 6 or 16 elements in it.

    If 6 elements are specified (a, b, ..., f), the matrix will be a 2D matrix constructed like:

    ace
    bdf
    001

    m11m12m13m14
